# projen-nvm

Helpers for Projen projects that use NVM.

## Docs

See [projen-nvm API Docs](https://dkershner6.github.io/projen-constructs/projen-nvm)

## Usage

Install the module:

```typescript
devDeps: ["projen-nvm"]
```

Import into your code:

```typescript
import { WhateverConstruct } from "projen-nvm";
```
